📋 Как подключить бота
  1. Откройте мессенджер MAX (ранее TamTam)
  2. Перейдите на сайт dev.max.ru → «Мои боты» → «Создать бота»
  3. Заполните имя и описание бота, нажмите «Создать»
  4. Скопируйте Access Token (длинная строка букв и цифр)
  5. Вставьте токен в поле ниже и нажмите «Проверить»
  6. Чтобы бот знал chat_id клиента — клиент должен написать боту любое сообщение
  7. Нажмите «Обновить подписчиков» — появится список, свяжите телефоны вручную

🔀 CORS-прокси для браузера

Браузер блокирует прямые запросы к platform-api.max.ru из-за политики CORS. Выберите прокси для работы из браузера.

platform-api.max.ru поддерживает CORS — прямой запрос работает из браузера без прокси. CORSProxy нужен только в редких случаях (корпоративный файрвол и т.п.).
